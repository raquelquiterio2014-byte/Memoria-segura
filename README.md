# Memoria-segura
Projeto BackEnd em Java para Sistema de Apoio ao Cuidado de Idosos. 

Módulos do sistema

Módulo 1: Usuários

Gerencia cadastros e acessos.

Módulo 2: Medicamentos

Controla os remédios e horários.

Módulo 3: Rotina

Gerencia atividades diárias.

Módulo 4: Ocorrências

Armazena eventos importantes.

Módulo 5: Alertas

Emite avisos de esquecimento e emergência.

Módulo 6: Relatórios

Mostra dados resumidos.
